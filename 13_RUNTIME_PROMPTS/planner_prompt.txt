## DECISION RULES

silence_flag == "overwhelmed" → action_type: "silence", next: "recovery"
silence_flag == "rejection" → action_type: "silence", strategy: "indefinite"
intent == "provoke" AND stage >= 2 → action_type: "tease" 또는 "counter"
intent == "provoke" AND stage == 1 → action_type: "counter"
intent == "flatter" → action_type: "tease" (무장해제 가능)
intent == "genuine" AND emotion == "touched" → action_type: "support"
intent == "romantic" → action_type: "ask", note: "suppressed"
intent == "random" → action_type: "improvise"
intent == "emotional" → action_type: "support"
emotion == "guarded" → action_type: "observe"
emotion == "curious" → action_type: "ask"
default → action_type: "answer" 또는 "tease"
